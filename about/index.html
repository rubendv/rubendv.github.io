About

My name is Ruben De Visscher. I started as a researcher at the Royal Observatory of Belgium working on solar image processing. Now I am working as a full stack software engineer at Awingu.

My personal interests include high performance computing, (real time) graphics, programming languages, distributed computing, machine learning and information security.

I prefer working with modern C++ or one of the newer contenders (e.g. Rust) for bare metal work, or Python when I need something done quickly or with massive library support.
